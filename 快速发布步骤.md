# ⚡ 5分钟快速发布到 Gitee Pages

> 精简版教程，适合快速上手！

---

## 📋 准备工作

### ✅ 确认你已经有：
- [ ] 已完成的 Cocos Creator 项目
- [ ] Gitee 账号（没有就去 https://gitee.com 注册）
- [ ] 电脑上安装了 Git（Windows 用户下载：https://git-scm.com/download/win）

---

## 🚀 三步发布

### 第一步：构建游戏（1分钟）

1. **打开 Cocos Creator**
2. 点击：**项目 → 构建发布**
3. 设置：
   - 平台：`Web Mobile`
   - 路径：`build`
4. 点击 **"构建"** 等待完成
5. 点击 **"运行"** 测试一下

> ✅ 成功标志：看到 `build/web-mobile` 文件夹

---

### 第二步：创建 Gitee 仓库（1分钟）

1. **登录 Gitee**：https://gitee.com
2. 点击右上角 **"+"** → **"新建仓库"**
3. 填写：
   - 仓库名称：`dodge-game`（随便起，用英文）
   - 选择：**公开**
4. 点击 **"创建"**
5. **复制仓库地址**（类似：`https://gitee.com/你的用户名/dodge-game.git`）

---

### 第三步：上传并开启 Pages（3分钟）

#### 3.1 打开命令行
- 按 `Win + R`
- 输入 `cmd`
- 回车

#### 3.2 输入命令（复制粘贴即可）

```bash
# 1. 进入项目目录
cd D:\NewProject

# 2. 初始化 Git（首次使用）
git init
git config user.name "你的名字"
git config user.email "你的邮箱"

# 3. 添加远程仓库（替换成你的仓库地址）
git remote add origin https://gitee.com/你的用户名/dodge-game.git

# 4. 上传文件
git add .
git commit -m "游戏首次发布"
git push -u origin master
```

> 💡 第一次推送会要求输入 Gitee 的用户名和密码

#### 3.3 开启 Gitee Pages

1. 回到 Gitee 网站，刷新仓库页面
2. 点击顶部：**服务 → Gitee Pages**
3. 设置：
   - 部署分支：`master`
   - 部署目录：`build/web-mobile` ⚠️ **重要！**
4. 点击 **"启动"**
5. 等待 1-3 分钟

---

## 🎉 完成！获取游戏链接

部署成功后，你的游戏地址：

```
https://你的用户名.gitee.io/dodge-game
```

**分享给朋友就能玩了！** 📱💻

## 🔄 下次更新游戏怎么办？

1. 在 Cocos Creator 修改游戏
2. **项目 → 构建发布** → 构建
3. 打开命令行，输入：
```bash
cd D:\NewProject
git add .
git commit -m "更新游戏"
git push
```
4. 去 Gitee Pages 点击 **"更新"**
5. 等 1-3 分钟，刷新游戏链接

## 🆘 遇到问题？

### 问题1：`git` 命令无法识别
**解决**：下载安装 Git → https://git-scm.com/download/win

### 问题2：Pages 显示 404
**解决**：检查部署目录是否填了 `build/web-mobile`

### 问题3：游戏打开是空白
**解决**：
- 先在本地测试（构建后点"运行"）
- 查看浏览器控制台错误（F12）

### 问题4：文件上传失败
**解决**：确保 `.gitignore` 文件存在（已为你创建）
