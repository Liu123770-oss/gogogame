# 🎮 躲避障碍物游戏

一个用 Cocos Creator 3.8 制作的刺激小游戏！

## 🎯 游戏特色

- 🚀 **动态难度**：第 8 秒速度突然从 3 暴增到 15！
- 🔥 **持续加速**：之后每秒增加 2，最高速度 50！
- ⚡ **极限挑战**：看你能坚持多少秒！

## 🎮 操作方式

- **触摸/鼠标**：点击屏幕左右两侧移动小球
- **键盘**：A/D 或 左右方向键移动

## 🏆 挑战等级

| 存活时间 | 等级 |
|---------|------|
| 8-10秒 | ⭐⭐ 入门 |
| 10-12秒 | ⭐⭐⭐ 熟练 |
| 12-15秒 | ⭐⭐⭐⭐ 高手 |
| 15-20秒 | ⭐⭐⭐⭐⭐ 大师 |
| 20秒+ | 💀 传奇 |
| 28秒+ | ⚡ 神级 |
